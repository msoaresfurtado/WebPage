def autocorr(x):
    result = np.correlate(x-np.mean(x), x-np.mean(x), mode='full')
    return result[int(np.floor(result.size/2)):]

So the general idea is that we want to find the maximum in the ACF (after the first dip) and identify the location on the x-axis. And while the actual value of the ACF is less important than the position of the peak, the ACF value can indicate whether the periodicity is strong (more periodic should have higher ACF values)

So this is point where my method may not be 100% correct, but I've been using scipy's signal tools to identify the peaks and it seems to have worked well enough so far. I'm still working on other ways to potentially identify local maxima so this may just be a starting point if you want to use a different method. Here's the functions I use though, and the 'gfpeaks' function is for the label on the plot so you can clearly see the values of the points.

# You'll also want to change the conversion here
def gfpeaks(p):
    l=[]
    for i in range(len(p)):
        l.append(r"acf: %5.4f" %acf[p[i]] +"     %5.4f" %(np.arange(len(acf))*(30/60/24))[p[i]]+' days')
    butFirst = l[0:]
    S = "\n".join(butFirst)
    return S

The trickiest part of the find_peaks function is choosing good height and distance values to use, which will vary based on your data. I have been using the maximum of the acf divided by some number (because we want to ignore that first spike) which usually will give me either one or a few peaks but you may need to play with those until it gives you a peak. Before that I was basically just doing guess and check until it gave me something becuase the distance isn't based on the actual x values.

Now for phase folding I wrote this so that if there's multiple peaks from the find_peaks function it can add subplots for comparison purposes. This may also need to be modified depending on your number of peaks or how you want to display the phase folded curve.

In [13]:
if peaks.shape[0]>1:
    num=peaks.shape[0]
    fig=plt.figure(figsize=[14,8])
    fig.suptitle('Phase folding',y=0.95)
    for i,v in enumerate(range(len(peaks))):
        fig.add_subplot(num,2,i+1) 
        plt.scatter(df.time%days[peaks[v]],df.flux,c=df.time,s=num)
        plt.subplots_adjust(hspace=0.5,wspace=0.3)
        plt.title( r" %5.4f" %days[peaks][v]+' days')
else:
    plt.scatter(df.time%days[peaks[0]],df.flux,c=df.time,s=4)
    plt.title( r"Phase folding with period = %5.4f" %days[peaks[0]]+' days')
